Descripción y Análisis de tu programa

Para este programa se genero un arreglo bidimensional que contenga las ventas mensuales de cada departamento que se indico (Ropa, Deportes, Jugueteria). Este programa sera de utilidad para aquella persona que tenga su negocio y quiera llevar un control de sus ganancias de cada articulo al mes.

Ademas de elaborar lo anterior mencionado, se nos indico implementar 3 metodos que nor permita hacer ciertas acciones.

1) Metodo para insertar elementos en el arreglo.
   Este primer metodo consiste en que nos deja agregar la cantidad monetaria que se obtuvo de ventas en un departamento y en que mes.

2) Metodo que permita, buscar algun elemento en particular.
   Este segundo metodo nos permite realizar una busqueda, nos permite realizar una consulta de la cantidad monetaria que se obtuvo en un determinado departamento y en que mes.
   
3) Metodo que permita, eliminar una venta en particular de algun departamento.
   Este tercer metodo nos permite eliminar una venta especifica de un departamento, este metodo es de gran ayuda, por si se registro una cifra incorrecta o algun otro caso.

Cuando ejecute el programa use el primer metodo para registrar ganancias en 3 departamentos, en el mismo mes. Despues use el segundo metodo para buscar una cantidad monetaria especifica (Funciono el metodo sin problemas). Y para el ultimo metodo elimine una cantidad especifica y listo.
